# UST2AUPLABEL
USTからAudacityのラベルトラック用テキストファイルを生成するツール（言w言）

## Dependency
Python 3.9.13

* utaupy

## Setup
ZIPファイルを解凍するだけ。になる予定。

## Usage
ust2auplabel.pyと同じフォルダに「fanta.ust」を置いて実行すると<br>
ノート通りにAudacity用のラベルが生成されます。
<br>
実行ファイルにUSTをドラッグアンドドロップするとAudacity向けのラベルトラックが生成されるようになる予定。
